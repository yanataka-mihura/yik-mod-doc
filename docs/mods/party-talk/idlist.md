# 使用できるかもしれないTopicId一覧

## TopicIdについて

TopicIdはModが特定タイミングでこのTopicIdで電波上に発言を流そうとした際、対応するTpicIdが定義上にあればそれを用いて発言します。

このTopicIdはModが定義したものと、ゲーム内で使用されているものの2種類があります。
