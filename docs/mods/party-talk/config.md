# BepInExの設定ファイル

## 設定ファイルについて

この設定ファイルは、Mod を導入し有効にした後にゲームを起動すると自動で作成されます。

通常は、初回起動後に以下のような場所へ作成されます。

```text
~\Steam\steamapps\common\Elin\BepInEx\config\yik.PartyTalk.cfg
```

設定を変更したい場合は、一度ゲームを終了してから設定ファイルを編集し、再度ゲームを起動してください。

## 一覧

| 項目                       | 種類      |     初期値 | 説明                                      |
| ------------------------ | ------- | ------: | --------------------------------------- |
| `CanNerunTalk`           | Boolean |  `true` | ネルンをパーティーメンバー扱いとして会話対象にするかを設定します。       |
| `ShowDebugLog`           | Boolean | `false` | 開発用のデバッグログを表示するかを設定します。                 |
| `MaxLines`               | Int32   |     `3` | 電波領域に表示する最大行数を設定します。                    |
| `NoCreateSample`         | Boolean | `false` | サンプルファイルを自動作成しないかを設定します。                |
| `NoChangePortrate`       | Boolean | `false` | ポートレート差分の適用を無効にするかを設定します。               |
| `CharacterTopicCooldown` | Int32   |     `0` | キャラクター単位で会話トピックのクールダウンを設定します。           |
| `PartyTopicCooldown`     | Int32   |     `0` | パーティー共通で会話トピックのクールダウンを設定します。            |
| `ShowExcelWarning`       | Boolean |  `true` | 旧仕様 Excel の会話定義を読み込んだ場合に警告を表示するかを設定します。 |
| `NoLoadBasePartyTalk`    | Boolean | `false` | PartyTalk 本体の会話定義を読み込まないようにするかを設定します。   |


---

## 項目説明

### `CanNerunTalk`

ネルンをパーティーメンバーとして扱いとしておしゃべり会話対象にするかどうかを設定します。

| 値       | 動作              |
| ------- | --------------- |
| `true`  | ネルンが会話対象になります。  |
| `false` | ネルンは会話対象になりません。 |

有効な場合、`PocketInNerun`存在しなければ`Nerun`（旧Excel用）を使用して参加します。

```ini
CanNerunTalk = true
```


---

### `ShowDebugLog`

開発用のデバッグログを表示するかどうかを設定します。

通常プレイでは `false` のままで問題ありません。  
不具合確認や開発中の動作確認をしたい場合に `true` にします。  
とてもうるさいので開発のたびに減ったり増えたりします。

| 値       | 動作             |
| ------- | -------------- |
| `true`  | デバッグログを表示します。  |
| `false` | デバッグログを表示しません。 |

```ini
ShowDebugLog = false
```

---


### `MaxLines`

電波領域に表示する最大行数を設定します。

ゲーム本来の未指定状態は `0` です。  
表示行数を制限したい場合は、任意の数値を指定します。  
指定数を超えた電波はすぐ消え始めます。

```ini
MaxLines = 3
```


---

### `NoCreateSample`

サンプルファイルを自動作成するかどうかを設定します。

Mod本体内のPartyTalk.jsonが存在しないとき、SamplePartyTalk.jsonからPartyTalk.jsonを作成させないためのスイッチです。  
`true` にすると「作らない」動作になります。

| 値       | 動作               |
| ------- | ---------------- |
| `true`  | サンプルファイルを作成しません。 |
| `false` | サンプルファイルを作成します。  |

```ini
NoCreateSample = false
```

---


### `NoChangePortrate`

ポートレート差分を適用するかどうかを設定します。

`true` にすると「変更しない」動作になり、会話定義でポートレート変更が指定されていても無視します。

| 値       | 動作               |
| ------- | ---------------- |
| `true`  | ポートレート差分を適用しません。 |
| `false` | ポートレート差分を適用します。  |

```ini
NoChangePortrate = false
```


---

### `CharacterTopicCooldown`

キャラクター単位で、同じ会話トピックが再び発言されるまでのクールダウンを設定します。

キャラクター指定系の発言にのみ使用されます。  
そのキャラクターが発言するとこの値が設定され、値が0になるまでの間は発言しなくなります。  
プレイヤーが1ターン行動するごとに、クールダウン値が1ずつ減少します。

`0` の場合はクールダウンなしです。

例えばPCの10倍速で動いてとても騒がしいときとかに1を指定します。

```ini
CharacterTopicCooldown = 0
```


---

### `PartyTopicCooldown`

パーティー共通で、同じ会話トピックが再び発言されるまでのクールダウンを設定します。

キャラクター指定系の発言にのみ使用されます。
プレイヤーが1ターン行動するごとに、クールダウン値が1ずつ減少します。

`0` の場合はクールダウンなしです。

```ini
PartyTopicCooldown = 0
```


---

### `ShowExcelWarning`

PartyTalk 本体が旧仕様 Excel の会話定義を読み込んでいる場合に、警告を表示するかどうかを設定します。

| 値       | 動作         |
| ------- | ---------- |
| `true`  | 警告を表示します。  |
| `false` | 警告を表示しません。 |

```ini
ShowExcelWarning = true
```

---

### `NoLoadBasePartyTalk`

PartyTalk 本体の会話定義を読み込むかどうかを設定します。

`true` にするとMod本体のPartyTalk.jsonを「読み込まない」動作になります。

| 値       | 動作                         |
| ------- | -------------------------- |
| `true`  | PartyTalk 本体の会話定義を読み込みません。 |
| `false` | PartyTalk 本体の会話定義を読み込みます。  |

```ini
NoLoadBasePartyTalk = false
```
