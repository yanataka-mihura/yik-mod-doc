# 特殊なテキスト変換

会話定義のText部分には`#`から始まる特定のワードを入れることにより、変換したテキストに差し替えられます。

例えば電波の表示領域は狭いので改行を間にいれて2段にしたり、プレイヤーの名前を叫ばせたりできます。

## Elin本体による変換

以下はElin本体により変換されます。ほかにもありますが似たり寄ったりだったり用途に合わないものなので割愛しています。

Elin本体の更新により内容が変更されたり増えたりすることがあり得ます。

| 指定文字 | 概要 |
| ----- | ----- |
| #god | 信仰している神 |
| #player | プレイヤーの二つ名 |
| #title | プレイヤーの二つ名 |
| #zone | 現在いる場所の名前 |
| #race | プレイヤー種族名 |
| #pc_full | プレイヤーフルネーム |
| #bigdaddy | "Mr.Adams" |
| #festival | 現在いる場所の祭りの名前。祭りじゃない場合も祭り |
| #brother2 | pcの性別により"お兄ちゃん" "お姉ちゃん" |
| #brother | pcの性別により"お兄さん,兄ちゃん,坊ちゃん" "お姉さん,姉ちゃん,お嬢ちゃん" からランダム |
| #onii2 | pcの性別により"お兄" "お姉" |
| #onii2 | pcの性別により"お兄ちゃん,お兄ちゃん♪,お兄ちゃーん,お兄ちゃん！,お兄ちゃん？,お兄ちゃん！？"または姉。 妹の鳴き声用 |
| #gender | pcの性別 |
| #he | pcの性別により"彼" "彼女" |
| #pc | pcの二つ名のないシンプルな名前 |
